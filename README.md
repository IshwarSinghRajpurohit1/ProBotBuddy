## Overview
ProBotBuddy is a backend-only project management and ticket tracking system
built using Spring Boot and PostgreSQL.

## Features
- Role-based login (Admin / User)
- Project & Module management
- Chatbot-based ticket generation
- Auto-generated ticket codes
- Ticket status tracking
- RESTful APIs

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA / Hibernate
- PostgreSQL
- REST APIs

## Architecture
- Controller Layer
- DTO Layer
- Repository Layer
- Entity Layer

## Database Design
- User
- Project
- Module
- Ticket

## How It Works
1. User logs in
2. Project is created
3. Modules are added
4. User selects module via chatbot
5. Ticket is generated with unique code
6. Status can be tracked

- JWT authentication
